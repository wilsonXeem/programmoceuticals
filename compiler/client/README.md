# CTD Compiler

A pharmaceutical dossier compilation tool for organizing and compiling CTD (Common Technical Document) format submissions.

## For Pharmaceutical Industries

This application helps pharmaceutical companies compile their product dossiers according to CTD format requirements.

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The application will open at [http://localhost:3000](http://localhost:3000)

## Building for Production

```bash
npm run build
```

## Features

- CTD structure-based file organization
- Drag-and-drop file upload
- Automatic folder structure generation
- ZIP file compilation
- Progress tracking
